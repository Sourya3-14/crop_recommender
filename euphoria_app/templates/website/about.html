About The Project

Bridging the gap between agricultural traditions and data science.

What We Are Doing

Our Crop Recommendation System is an intelligent platform designed to help modern farmers make informed, data-backed decisions without scientific knowledge. By assessing multiple variables like soil content and micro-climates, the system eliminates traditional guesswork, ensuring that resources are only spent on crops that have a high probability of thrives.

How Machine Learning is Used

We use predictive classification algorithms (such as K Nearest Neighbors, Logistic Regression, Random Forest Classifier, Naive Bayes, Support Vector Machines, Artificial Neural Network ) trained on comprehensive multi-regional agricultural datasets and compared the algorithms using k-fold cross validation score .

The model dynamically processes 7 key features: Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, Soil pH, and Rainfall to accurately predict the crop most suited for the environment.
